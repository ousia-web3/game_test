# 명예의 전당 3자리 이니셜 입력 시스템 작업 계획

사용자의 요청에 따라 텍스트 입력 방식이었던 닉네임 등록 시스템을 아케이드 스타일의 **3자리 알파벳 선택 방식**으로 전환합니다. 키보드 화살표 키를 이용해 각 자릿수의 문자를 변경하고 이동할 수 있도록 구현합니다.

## 주요 작업 내용

1.  **HTML 구조 변경 (`index.html`)**:
    *   기존 `<input type="text">`를 제거합니다.
    *   3개의 문자 슬롯(`char-slot`)을 가진 입력 컨테이너를 추가합니다.
    *   현재 선택된 위치를 나타낼 수 있는 구조를 설계합니다.

2.  **CSS 스타일링 (`styles.css`)**:
    *   아케이드 게임 느낌의 큼직한 문자 박스 스타일을 적용합니다.
    *   현재 활성화된 슬롯에 테두리 애니메이션 또는 강조 효과를 추가합니다.
    *   상/하 화살표 가이드를 시각적으로 제공하여 조작 가능함을 직관적으로 보여줍니다.

3.  **입력 로직 구현 (`app.js`)**:
    *   **상태 관리**: 현재 선택된 슬롯 인덱스(0, 1, 2)와 각 슬롯의 문자 데이터(A-Z)를 관리합니다.
    *   **키보드 이벤트 핸들링**:
        *   `ArrowUp` / `ArrowDown`: 현재 슬롯의 문자를 순환 (A-Z-0-9 등).
        *   `ArrowLeft` / `ArrowRight`: 슬롯 간 이동 (0 ↔ 1 ↔ 2).
        *   `Enter`: 현재 완성된 3자리 이니셜로 점수 등록.
    *   **결과 화면 노출 시 활성화**: 게임 종료 후 결과 화면이 나타날 때만 해당 조작이 활발하게 동작하도록 제어합니다.

## 상세 수정 계획

| 파일 | 수정 내용 |
| :--- | :--- |
| `index.html` | `.rank-form` 내부를 3개의 문자 슬롯 구조로 교체 |
| `styles.css` | `.char-entry`, `.char-slot`, `.active-slot` 등 관련 스타일 정의 |
| `app.js` | `initNameEntry`, `handleNameInput` 메소드 추가 및 `submitScore` 연동 |

## 기대 효과
*   엔들리스 리듬 게임에 어울리는 아케이드 감성의 UX를 제공합니다.
*   키보드 화살표만으로 모든 게임 과정을 완료할 수 있어 조작의 일관성이 향상됩니다.
