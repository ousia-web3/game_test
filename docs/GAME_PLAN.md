# 용사의계단 게임 기획

## 한 줄 정의
생명 코인을 주입해 출정한 용사가 무너지는 계단과 적을 피해 최대한 오래 상승하며 점수를 올리는 아케이드 액션 게임.

## 코어 판타지
- 플레이어는 출정 전 생명 코인을 심장 코어에 주입한다.
- 게임이 시작되면 용사는 자동으로 전진하고, 플레이어는 좌우 이동과 타이밍으로 생존을 이어 간다.
- 계단을 오를수록 속도와 압박이 올라가며, 생명 수치 관리가 핵심이 된다.
- 종료 후 닉네임과 점수를 등록해 리더보드 경쟁을 만든다.

## 핵심 재미
- 시작 직전의 "생명 주입" 의식감
- 한 번 더 버티면 점수가 오르는 즉시성
- 좌우 회피와 아이템 획득의 짧고 강한 판단
- 즉시 기록 등록으로 이어지는 경쟁 루프

## MVP 범위
- 메인 인트로 화면
- 생명 코인 주입 UI
- 3레인 기반 계단 오르기 플레이
- 생명, 점수, 거리, 속도 시스템
- 코인 획득과 피해 판정
- 게임오버 연출
- 점수 등록 API
- 로컬 리더보드 조회

## 아트 디렉션
- 다크 판타지와 아케이드 네온의 혼합
- 붉은 심장 코어, 황동 코인, 청록 에너지 계단
- 픽셀 감성과 현대적 UI를 혼합한 메인 화면

## 확장 아이디어
- 보스 계단 구간
- 스킬 카드
- 일일 미션
- 실제 계정 기반 온라인 랭킹
