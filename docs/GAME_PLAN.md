# 영웅의 계단 게임 기획 (Rhythm Action)

## 한 줄 정의
8비트 사운드트랙의 비트에 맞춰 떨어지는 계단 노트를 처리하며 생존하고 최고 기록을 달성하는 **리듬 아케이드 액션 게임**.

## 코어 판타지
- 플레이어는 리듬의 기운을 심장 코어에 주입하여 전장에 나선다.
- 음악의 비트가 용사의 발판이 되며, 리듬을 놓치면 계단이 무너지고 생명이 깎인다.
- 레벨이 올라갈수록 빨라지는 템포와 복잡한 비트 속에서 성취감을 느낀다.

## 핵심 재미
- **비트 동기화**: 8비트 사운드와 조작이 일치할 때의 타격감.
- **콤보 스릴**: 연속 성공으로 점수가 기하급수적으로 오르는 쾌감.
- **다양한 음원**: 레벨별로 달라지는 10곡의 고유 멜로디.
- **즉각적인 경쟁**: 게임 종료 후 바로 이어지는 랭킹 등록 시스템.

## MVP 범위
- 8비트 사운드 생성기 및 시퀀서 구현
- 3레인 기반 노트(계단) 낙하 및 판정 시스템
- 4단계 판정(Perfect/Great/Good/Miss) 및 콤보 로직
- 상위 10개 레벨 디자인 (BPM 80 ~ 170)
- 생명 코인 주입 UI 및 결과 시스템
- SQLite 기반 온라인 리더보드 API

## 아트 디렉션
- 다크 판타지 + 레트로 네온 스타일.
- **청록색 에너지 계단**: 판정선에 닿을 때 빛나는 효과.
- **심장 코어**: 리듬에 맞춰 박동하는 UI 요소.
- **8비트 사운드**: 화려한 악기 대신 투박하지만 매력적인 사각형파(Square Wave) 멜로디.

## 확장 아이디어
- 유저 제작 비트(Custom Map) 기능
- 특수 롱 노트 및 동시 치기 노트 추가
- 캐릭터 스킨 및 고유 스킬 시스템
