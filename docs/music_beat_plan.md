# 레벨별 음악 비트 다양화 작업 계획

사용자의 요청에 따라 게임 플레이 중 각 단계(레벨 1~10)마다 서로 다른 음악 비트와 멜로디를 제공하도록 시스템을 개선합니다.

## 주요 작업 내용

1.  **AudioManager 클래스 확장**:
    *   현재 3개인 `this.tracks` 리스트를 10개로 확장합니다.
    *   각 트랙(Lv.1 ~ Lv.10)에 고유한 8비트 스타일의 멜로디와 리듬 시퀀스를 정의합니다.
    *   초반 레벨은 밝고 단순하게, 후반 레벨로 갈수록 긴박하고 복잡한 구성을 가집니다.

2.  **BGM 재생 로직 수정 (`startBGM`)**:
    *   레벨 범위에 따라 3개 트랙을 공유하던 로직을 삭제합니다.
    *   전달받은 `level` 값을 인덱스로 사용하여 (`level - 1`) 정확히 해당 레벨의 트랙을 재생하도록 변경합니다.

3.  **8비트 멜로디 구성 전략**:
    *   **Lv.1~3**: 밝은 분위기, 장조(Major) 멜로디, 단순한 베이스라인.
    *   **Lv.4~6**: 템포 상승, 당김음(Syncopation) 추가, 기술적인 아르페지오 포함.
    *   **Lv.7~9**: 긴장감 고조, 단조(Minor) 및 반음계적 구성, 강한 비트감.
    *   **Lv.10**: 최종 보스 분위기, 빠르고 화려한 피날레 멜로디.

## 상세 수정 계획

| 파일 | 수정 위치 | 내용 |
| :--- | :--- | :--- |
| `app.js` | `AudioManager.constructor` | `this.tracks` 배열에 10개의 고유 트랙 데이터 추가 |
| `app.js` | `AudioManager.startBGM` | 레벨별 트랙 선택 로직 업데이트 (`trackIdx = level - 1`) |

## 기대 효과
*   각 레벨을 클리어할 때마다 새로운 배경음악이 제공되어 플레이어의 몰입감과 도전 욕구를 자극합니다.
*   시각적 속도 증가와 청각적 비트 변화가 맞물려 더욱 완성도 높은 리듬 게임 경험을 제공합니다.
