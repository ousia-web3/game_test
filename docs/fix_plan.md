# UI 및 렌더링 버그 수정 계획

재미니님, 요청하신 두 가지 문제를 해결하기 위한 계획입니다.

## 1. 인트로 화면 UI 중앙 정렬 수정
- **문제 원인**: 인트로 화면의 `.content` 영역 내에서 요소들이 중앙 정렬되지 않아 버튼과 기록 텍스트가 왼쪽으로 치우쳤습니다.
- **해결 방안**: `styles.css`를 수정하여 컨텐츠 영역을 플렉스 박스로 설정하고 중앙 정렬을 강제합니다.

## 2. 게임 플레이 화면 정보 표시 수정
- **문제 원인**: 게임 시작 시 캔버스 크기가 0으로 잡히는 버그로 인해 렌더링이 되지 않았습니다.
- **해결 방안**: `app.js`에서 게임 시작 직후 리사이즈 로직을 실행하여 캔버스를 활성화합니다.

## 작업 순서
1. `public/styles.css` 수정: 중앙 정렬 누락 스타일 추가
2. `public/app.js` 수정: `startGame` 시 `resize` 호출 추가
3. 결과 확인 및 안내
