# Stairs of the Hero (영웅의 계단)

영웅의 계단은 복고풍 8비트 사운드와 함께 리듬에 맞춰 계단 노트를 처리하는 **리듬 액션 게임**입니다.

## 🎮 게임 개요
사용자는 각 레벨별로 제공되는 고유한 8비트 배경음악의 비트에 맞춰 떨어지는 계단(노트)을 정확한 타이밍에 입력해야 합니다. 단순한 계단 오르기가 아닌, 음악과 비트가 중심이 되는 리듬 조작 게임입니다.

## ✨ 주요 기능
- **리듬 기반 조작**: 8비트 음악 비트에 맞춰 떨어지는 노트를 판정선에 맞춰 입력합니다.
- **레벨별 고유 트랙**: Lv.1부터 Lv.10까지 각각 다른 멜로디와 BPM(80~170)을 가진 10개의 트랙이 제공됩니다.
- **판정 시스템**: Perfect, Great, Good, Miss의 4단계 판정 시스템과 콤보에 따른 점수 배율 시스템이 적용됩니다.
- **생명 시스템**: 시작 전 설정한 생명 코인에 따라 생명력이 결정되며, Miss 판정 시 생명이 감소합니다.
- **리더보드**: 최종 점수를 기록하고 다른 플레이어와 경쟁할 수 있습니다.

## 🛠 기술 스택
- **Frontend**: HTML5, Vanilla CSS, JavaScript (Web Audio API 기반 자체 시퀀서 구현)
- **Backend**: Node.js (Express 프레임워크 스타일)
- **Database**: SQLite3 (점수 저장 및 조회)

## 📁 프로젝트 구조
- `public/`: 웹 프론트엔드 (리듬 판정 로직, 시각 효과, 오디오 관리)
- `data/`: 리더보드 데이터베이스 저장소
- `docs/`: 리듬 게임 기획 및 음악 비트 작업 계획 문서
- `server.js`: API 서버 및 정적 파일 서빙

## 🚀 시작하기

### 사전 준비
- [Node.js](https://nodejs.org/) 설치 필수

### 설치 및 실행
1. 의존성 설치: `npm install`
2. 서버 실행: `npm start`
3. 브라우저에서 `http://localhost:3000` 접속

## 🕹 조작 방법 (Rhythm Control)
- **레인 이동 및 판정**: 
    - `A`, `S`, `D` 키 또는 `왼쪽`, `아래`, `오른쪽` 방향키
    - 각각 왼쪽, 중앙, 오른쪽 레인의 노트를 처리합니다.
- **콤보 유지**: 노트를 놓치지 않고 연속으로 맞추면 높은 점수를 획득할 수 있습니다.
